# Motor Controller

you will need to install socketio

```
pip install "python-socketio[client]"
```

or depending on how your setup works

```
pip3 install "python-socketio[client]"
```

run the server with:

```
python3 ./app.py
```

If it crashes and needs to be restarted during development
use.

```
 py .\forever.py
```
